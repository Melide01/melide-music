# Documentation du projet audio

Ce dossier contient la documentation du fichier `data/audio_template.json`.

## Contenu

- `audio_project_structure.md` : description de la structure globale du projet audio.
- `sections_and_tracks.md` : description des sections, pistes et notes.

## Objectif

Ce JSON représente un projet audio complet avec :

- un tempo global (`tempo`)
- des instruments définis dans `instruments`
- des sections musicales dans `sections`
- des pistes et des événements de notes pour chaque instrument

Les noms d'instruments et les clés JSON restent en anglais dans la documentation.
